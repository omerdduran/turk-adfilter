# Kurulum Rehberi

> Turk-AdFilter'ı kullanmak için aşağıdaki RAW bağlantısını kullanabilirsiniz:
> 
> ```bash
> https://raw.githubusercontent.com/omerdduran/turk-adfilter/main/turk-adfilter.txt
> ```

## uBlock Origin'de Kullanım

1. **Ayarları Açın**: uBlock Origin ayarlarını açın
2. **Filtreler Sekmesi**: "Filtreler" sekmesine tıklayın
3. **İçe Aktarın**: "Özel" bölümünde "İçe Aktar" butonuna tıklayın
4. **Bağlantıyı Ekleyin**: Yukarıdaki RAW bağlantısını yapıştırın ve "Değişiklikleri Uygula" butonuna tıklayın

![uBlock Origin Kurulum](/assets/ublock-kurulum.png)

## AdGuard'da Kullanım

1. **Ayarları Açın**: AdGuard ayarlarını açın
2. **Filtreler Bölümü**: "Filtreler" > "Özel Filtreler" sekmesine gidin
3. **Filtre Ekle**: "Özel Filtre Ekle" butonuna tıklayın
4. **Bağlantıyı Ekleyin**: Yukarıdaki RAW bağlantısını yapıştırın ve "Ekle" butonuna tıklayın

## Pi-hole'da Kullanım

1. **Yönetim Paneline Giriş**: Pi-hole yönetim panelinize giriş yapın
2. **Adlist Ekleme**: "Group Management" > "Adlists" sekmesine gidin
3. **Liste Ekle**: "Add a new adlist" kısmına aşağıdaki Pi-hole bağlantısını yapıştırın
4. **Ekle ve Güncelle**: "Add" butonuna tıklayın ve ardından "Update Gravity" ile güncellemeleri uygulayın

> Pi-hole için kullanmanız gereken bağlantı:
> 
> ```bash
> https://raw.githubusercontent.com/omerdduran/turk-adfilter/refs/heads/main/hosts.txt
> ```

## Mobil Cihazlarda Kullanım

### Android

> Android cihazlarda reklamları engellemek için aşağıdaki uygulamaları kullanabilirsiniz:

- **AdGuard for Android**: Uygulamayı yükleyin ve yukarıdaki AdGuard talimatlarını takip edin
- **Blokada**: Uygulamayı yükleyin ve özel liste olarak RAW bağlantısını ekleyin
- **DNS66**: Uygulamayı yükleyin ve özel host kaynağı olarak RAW bağlantısını ekleyin

### iOS

> iOS cihazlarda reklamları engellemek için aşağıdaki yöntemleri kullanabilirsiniz:

- **AdGuard for iOS**: Uygulamayı yükleyin ve ayarlardan özel liste olarak RAW bağlantısını ekleyin
