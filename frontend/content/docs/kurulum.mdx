# Kurulum Rehberi

Turk-AdFilter'ı kullanmak için aşağıdaki RAW bağlantısını kullanabilirsiniz:

```
https://raw.githubusercontent.com/omerdduran/turk-adfilter/main/turk-adfilter.txt
```

## uBlock Origin'de Kullanım

1. uBlock Origin ayarlarını açın
2. "Filtreler" sekmesine tıklayın
3. "Özel" bölümünde "İçe Aktar" butonuna tıklayın
4. Yukarıdaki RAW bağlantısını yapıştırın ve "Değişiklikleri Uygula" butonuna tıklayın

![uBlock Origin Kurulum](/assets/ublock-kurulum.png)

## AdGuard'da Kullanım

1. AdGuard ayarlarını açın
2. "Filtreler" > "Özel Filtreler" sekmesine gidin
3. "Özel Filtre Ekle" butonuna tıklayın
4. Yukarıdaki RAW bağlantısını yapıştırın ve "Ekle" butonuna tıklayın

## NextDNS'de Kullanım

1. NextDNS kontrol panelinize giriş yapın
2. "Güvenlik" sekmesine gidin
3. "Reklam Engelleme Listeleri" bölümüne inin
4. "Özel Liste Ekle" seçeneğine tıklayın
5. Yukarıdaki RAW bağlantısını yapıştırın ve "Ekle" butonuna tıklayın

## Mobil Cihazlarda Kullanım

### Android

Android cihazlarda reklamları engellemek için:

1. **AdGuard for Android**: Uygulamayı yükleyin ve yukarıdaki AdGuard talimatlarını takip edin
2. **Blokada**: Uygulamayı yükleyin ve özel liste olarak RAW bağlantısını ekleyin
3. **DNS66**: Uygulamayı yükleyin ve özel host kaynağı olarak RAW bağlantısını ekleyin

### iOS

iOS cihazlarda reklamları engellemek için:

1. **AdGuard for iOS**: Uygulamayı yükleyin ve ayarlardan özel liste olarak RAW bağlantısını ekleyin
2. **NextDNS**: iOS'un DNS ayarlarını NextDNS'ye yönlendirin ve yukarıdaki NextDNS talimatlarını uygulayın

## Tarayıcıya Göre Kurulum

| Tarayıcı | Önerilen Eklenti | Kurulum |
|----------|------------------|---------|
| Chrome | uBlock Origin | Chrome Web Mağazası'ndan yükleyin ve yukarıdaki uBlock Origin talimatlarını uygulayın |
| Firefox | uBlock Origin | Firefox Eklentiler'den yükleyin ve yukarıdaki uBlock Origin talimatlarını uygulayın |
| Safari | AdGuard for Safari | App Store'dan yükleyin ve yukarıdaki AdGuard talimatlarını uygulayın |
| Edge | uBlock Origin | Edge Eklentiler'den yükleyin ve yukarıdaki uBlock Origin talimatlarını uygulayın |
| Opera | uBlock Origin | Opera Eklentiler'den yükleyin ve yukarıdaki uBlock Origin talimatlarını uygulayın |
