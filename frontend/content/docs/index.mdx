# Turk-AdFilter Projesi

Turk-AdFilter <span className='text-fd-muted-foreground text-sm'>(Türk Reklam Filtresi)</span> is a **topluluk tabanlı filtre listesi** designed to be fast, flexible,
and works seamlessly with popular ad blockers.

## Proje Hakkında

Bu proje, Türkçe internet kullanıcıları için özel olarak oluşturulmuş ve Türkiye'deki web sitelerinde ve uygulamalarda karşılaşılan reklamları, izleyicileri ve zararlı içerikleri engellemeyi amaçlar.
